# 買ってよかったものランキングアプリ

お気に入りの商品をランキング形式でまとめて共有できるWebアプリケーションです。

## 機能

- ✅ ランキングの作成・編集・削除
- ✅ 商品の追加・削除（最大20位まで）
- ✅ ドラッグ&ドロップによる順位の並べ替え
- ✅ ランキングのURL共有機能
- ✅ Amazon商品情報の検索（ASIN対応）
- ✅ モダンで美しいUI

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 使用方法

1. **ランキングの作成**
   - ホーム画面でランキングのタイトルを入力
   - 「新しいランキングを作成」をクリック

2. **商品の追加**
   - 「商品を追加」ボタンをクリック
   - ASINを入力して「検索」をクリックすると、Amazon商品情報を自動取得（オプション）
   - または、手動で商品名、画像URL、リンクを入力
   - 「追加」をクリック

3. **順位の変更**
   - 商品カードをドラッグ&ドロップして順位を変更

4. **商品の削除**
   - 商品カードの削除ボタン（ゴミ箱アイコン）をクリック

5. **ランキングの共有**
   - 「共有」ボタンをクリックしてURLをコピー
   - 共有されたURLでランキングを閲覧可能

## Amazon Product Advertising API統合

現在、Amazon APIはモック実装になっています。実際のAPIを使用するには：

1. [Amazon Product Advertising API 5.0](https://webservices.amazon.co.jp/paapi5/documentation/) に登録
2. APIキーとシークレットを取得
3. `app/api/amazon/route.ts` を更新して実際のAPIリクエストを実装

## 技術スタック

- **フロントエンド**: Next.js 14 (App Router), React, TypeScript
- **スタイリング**: Tailwind CSS
- **ドラッグ&ドロップ**: @dnd-kit
- **アイコン**: lucide-react
- **データベース**: インメモリ（本番環境では実際のDBに置き換え推奨）

## 注意事項

- 現在の実装ではデータはインメモリに保存されています。サーバーを再起動するとデータが失われます。
- 本番環境では、PostgreSQL、MySQL、MongoDBなどの永続的なデータベースを使用することを推奨します。
